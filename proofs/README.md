# DateSAT Verification

This repository contains mechanized formal semantics for DateSAT as well as machine-checkable proofs of well-formedness, monotonicity, and equivalence of DateSAT encodings (Naive a.k.a. Simple, Epoch-Based, Hybrid, Alpha-Beta).

First, [install Verus](https://github.com/verus-lang/verus/blob/main/INSTALL.md).

Then, in this repository, run `/path/to/verus src/main.rs`

Scroll down to the bottom of `src/main.rs` to see the main claims (theorems) being proved.
